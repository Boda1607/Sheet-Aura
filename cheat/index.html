Creates an array.

RegEx Cheat Sheet

Characters

Matches any single character (except newline).

.
`a.c` matches "abc", "axc", "a1c".

Matches any digit (0-9).

\d
`\d\d` matches "12", "05".

Matches any non-digit character.

\D
`\D\D` matches "ab", "$%".

Matches any whitespace character (space, tab, newline).

\s
`word\sword` matches "word word".

Matches any non-whitespace character.

\S
`\S\S` matches "Hi".

Matches any word character (alphanumeric and underscore).

\w
`\w+` matches "hello", "world123".

Matches any non-word character.

\W
`\W` matches "!", "@", "#".

Matches a literal special character (e.g., `.`, `*`, `+`).

\.
`abc\.xyz` matches "abc.xyz".

Quantifiers

Matches 0 or more occurrences of the preceding character/group.

*
`ab*c` matches "ac", "abc", "abbc".

Matches 1 or more occurrences of the preceding character/group.

+
`ab+c` matches "abc", "abbc", but not "ac".

Matches 0 or 1 occurrence of the preceding character/group.

?
`colou?r` matches "color", "colour".

Matches exactly `n` occurrences.

{n}
`\d{3}` matches "123".

Matches at least `n` occurrences.

{n,}
`a{2,}` matches "aa", "aaa", etc.

Matches between `n` and `m` occurrences (inclusive).

{n,m}
`\d{2,4}` matches "12", "123", "1234".

Makes the quantifier lazy (matches shortest possible string).

*?
`a.*?b` matches "a*b" in "axbcyb".

Anchors & Groups

Matches the beginning of a string.

^
`^Hello` matches "Hello World" but not "World Hello".

Matches the end of a string.

$
`World$` matches "Hello World" but not "World Hello".

Matches an exact sequence of characters literally.

\b
`\bword\b` matches "word" in "this word is", but not in "keyword".

Matches a non-word boundary.

\B
`\Bword\B` matches "word" in "forward", but not "word ".

Defines a character set (matches any one character within the brackets).

[abc]
`[aeiou]` matches any vowel.

Defines a character set (matches any one character NOT within the brackets).

[^abc]
`[^0-9]` matches any non-digit.

Groups multiple tokens together and creates a capturing group.

(ab|cd)
Matches "ab" or "cd". Captured content can be recalled.

Groups multiple tokens together without creating a capturing group.

(?:ab|cd)
Matches "ab" or "cd". Not captured.

Alternation (OR operator).

|
`cat|dog` matches "cat" or "dog".

Assertions & Flags

Positive lookahead: matches if `pattern` is followed by `lookahead`.

pattern(?=lookahead)
`foo(?=bar)` matches "foo" only if followed by "bar".

Negative lookahead: matches if `pattern` is NOT followed by `lookahead`.

pattern(?!lookahead)
`foo(?!bar)` matches "foo" only if NOT followed by "bar".

Global flag: matches all occurrences of a pattern, not just the first.

/pattern/g
Used with `String.prototype.matchAll()` or `RegExp.prototype.exec()`.

Case-insensitive flag: performs a case-insensitive match.

/pattern/i
`/hello/i` matches "Hello", "hello", "HELLO".

Multiline flag: `^` and `$` match the start/end of each line, not just the start/end of the string.

/pattern/m
Allows line-by-line matching.
